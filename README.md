# Bibliotecas Utilizadas
<div style="text-align: justify;">
<p>Durante o desenvolvimento do backend foram utilizadas uma gama de bibliotecas que podem ser consultadas dentro da pasta padrão node_modules, sendo adicionadas ao longo do projeto e as necessidades que foram surgindo. Irei citar aqui as principais e mais relevantes. </p>

<h2>Express.js</h2>
<p>O Express.Js é um Framework rápido e um dos mais utilizados em conjunto com o Node.Js, facilitando no desenvolvimento de aplicações back-end e até, em conjunto com sistemas de templates, aplicações full-stack.

<h2>Cors</h2>
<p>Recurso importante de segurança do navegador que restringe solicitações HTTP de origem cruzada com outros servidores e especifica quais domínios acessam seus recursos.

<h2>Bcrypt</h2>
<p>O BCrypt foi desenvolvido por Niels Provos e David Mazières com o propósito de esconder senhas criadas pelos usuários em forma de texto "puro" em dados indecifráveis, utilizando o algoritmo hash.</p>


<h2>JSON Web Tokens</h2>
<p>Seria um padrão para autenticação e troca de informações, possibilitando o armazenamento seguro de objetos JSON.</p>
</div>
